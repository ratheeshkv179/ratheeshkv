<a name="how-to-write-code"></a>
## Writing code using SDK
Import module from the .\src\intersight folder
```powershell
Import-Module .\src\intersight\intersight.psd1
```
Create IntersightApiClient
```powershell
New-IntersightApiClient $intersightUrl $private_key_path $api_key_id
```
1. $api_key_id : api key id value from Intersight UI
- Example : $api_key_id="5a61b9896736327a31bbebff/5a61b7586736327a31bbeb74/5a7c3054647339736e57b82c"
2. $private_key_path : location of .pem file saved for private key from Intersight UI
- Example : $private_key_path="C:\\Users\\ratkv\\source\\repos\\key.pem"
3. $intersightUrl : URL of Intersight
- Example : $intersightUrl = "https://ucs.cisco.com/api/v1"

for($i=0;$i -lt 5;$i++){
